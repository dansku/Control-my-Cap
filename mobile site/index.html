What is this?

This is crowd-sourced control for my mortar board. Pick a color to add it to the queue. The high-intensity LEDs mounted on my motar board will cycle through all the colors that have been chosen by the crowd. It will tweet at you when it gets to your color if you provide your twitter handle! Learn More!

How Does it work?

It's an experiment. Why? Because, WHY NOT? My graduation mortar board is sporting an array of 16 350mA high-intensity LEDs. When you pick a color above, it is added to queue of colors on a database that lives in the cloud. Under my robe, I'm equipped with a laptop battery pack, a tiny WiFi-connected linux computer, and a handheld display. The computer is constantly querying the cloud database server via its WiFi connection, and adding colors to the queue. It communicates to the custom LED printed circuit board on my mortar board using a USB-to-Serial cable. The LED board includes an 8-bit microcontroller that parses the commands from the linux computer, and adjusts the LEDs accordingly. Using my handheld display, I can see that status of the program, adjust brightness, change modes, etc. When your color is displayed for the first time, the linux computer will tweet at your twitter username, if you chose to provide it.
